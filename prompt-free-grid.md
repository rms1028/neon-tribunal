# 자유토론 UI 리디자인 — Pinterest 스타일 카드 그리드

프로젝트 루트에 있는 free-discuss-watcha.jsx 파일의 LayoutGrid 컴포넌트를 참고해서 자유토론 상세 페이지를 리디자인해줘.

## 핵심 컨셉
왓챠피디아/Pinterest처럼 프로필 카드 + 의견이 2열 그리드로 배치되는 구조.
기존 도구 & 분석 아이콘, 정렬 탭은 제거하고 미니멀하게.

## 전체 구조

### 1. 헤더 (sticky)
- 왼쪽: ← 홈 버튼 + 토론 제목 (fontSize 17, fontWeight 900)
- 제목 아래: "자유토론 · N개 의견 · N개 댓글" 부제 (fontSize 11, color #444)
- 오른쪽: "🔥 N 반응" 텍스트 + "✍️ 의견 쓰기" 버튼
- 배경: blur 반투명

### 2. 메인: 2열 카드 그리드
- CSS columns: "2 320px", columnGap: 16px
- padding: 20px 24px
- 카드가 내용 길이에 따라 높이가 다름 (Masonry 효과)

### 3. 의견 카드 디자인

각 카드 구조 (위에서 아래):
- 프로필 영역: 아바타(36px) + 이름 + 레벨 뱃지 + 시간
- 의견 본문: fontSize 14, color #aaa, lineHeight 1.75
- 액션 바: ♥ 좋아요 숫자 + 💬 댓글 숫자 (상단 border로 구분)

카드 스타일:
- padding: 18px 20px
- borderRadius: 18px
- 배경: rgba(255,255,255,0.02)
- 테두리: 1px solid rgba(255,255,255,0.04)
- hover 시: 테두리 밝아짐 + translateY(-2px) + 그림자
- break-inside: avoid (카드가 컬럼 사이에서 잘리지 않게)

프로필 아바타:
- 36px 사각 라운드 (borderRadius: 10px)
- 배경색: 사용자마다 다른 컬러 (초록, 핑크, 파랑, 노랑, 보라, 주황 중 순환)
- 이니셜 한 글자, fontWeight 800
- border: 1.5px solid 해당컬러30

좋아요 버튼:
- 클릭 시 ♡ → ♥ 전환 + 색상 #ff4d8d + 숫자 +1
- 다시 클릭하면 취소

### 4. 의견 쓰기
- 헤더의 "의견 쓰기" 버튼 클릭 시 모달 오픈
- 모달: 배경 blur + 중앙 카드 (540px)
- textarea + 등록/취소 버튼

## 제거할 기존 요소
- 도구 & 분석 아이콘 바 (🕐 📊 📈 ⚙️ ⭐)
- "뜨거운순 / 깊은 대화순 / 최신순" 정렬 탭
- 기존 의견 카드의 큰 사각형 디자인
- "첫 의견을 시작해보세요" 텍스트가 카드 안에 작게 있는 것
  → 의견 0개면 화면 중앙에 크게 빈 상태 표시

## 컬러
- 다크 테마 유지
- 좋아요: #ff4d8d
- 프로필 컬러: 사용자마다 다르게 순환
  #00e4a5, #ff4d8d, #55b3ff, #ffd055, #c084fc, #ff8a50
